RS Coming Soon plugin Creates a page for your Website while it's under construction and collects emails from your visitors and show in dashboard.


== Description ==

RS Coming Soon plugin has super admin panel where your customize anything including fonts and translate to any language and control saved emails from visitor and export to csv formate even you can remove emails from admin panel. Its fully SEO optimized under construction plugin


See live demo here: http://themebon.com/rs-coming-soon/

== Installation ==

Installing this plugin as regular wordpress plugin.

After install and activate, you will see on top to Redux Framrwork plugin required, install redux plugin as well. After that you can see a dashboard menu in bottom called <strong>RS Coming Soon & RS Subscriber</strong>. In <strong>RS Coming Soon</strong> you can customize all things and in <strong>RS Subscriber</strong> you can control your visitors emails.
